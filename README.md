# facade 1.0.0

> 采用nuxt + express构建的产品介绍网站，让你快速拥有一个可以快速展示自身产品或者项目的响应式网站

## 项目介绍
自身产品或者项目主要以卡片式ui介绍给用户，让客户快速了解你的全部产品或者项目

## 项目环境介绍
> iview4.0.2 + nuxt2.10.2 + express4.17.1
>
> 主要依赖为mysql 和 阿里云的oss对象存储
> 
> src 文件夹 => 前端代码
>
> server 文件夹 => 后台代码
>
> release 文件夹 => 需要用到sql文件，线上nginx.conf配置示例
