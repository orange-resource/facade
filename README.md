# facade

> 采用nuxt + express构建的产品介绍网站，让你快速拥有一个可以快速展示自身产品或者项目的响应式网站

## 项目环境介绍
> nuxt2.10.2 + express4.17.1 
> 
> src 文件夹 => 前端代码
>
> server 文件夹 => 后台代码
>
> release 文件夹 => 需要用到sql文件，线上nginx.conf配置示例
